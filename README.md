面向中层管理人员、专家岗、省代表等异地员工的信息采集与管理平台，支持在线问卷填报、审核工作流、邮件通知与定时发送、数据备份等完整业务闭环。

## 快速开始

**环境要求**：Python 3.8+

```bash
# 安装依赖
pip install -r requirements.txt

# 启动开发服务器
python main.py

# 生产部署（Linux/macOS）
gunicorn main:app -b 0.0.0.0:5495
```

服务默认监听 `http://0.0.0.0:5495`。

**环境变量**（可选，不配置则使用默认值）：

| 变量名 | 说明 | 默认值 |
|-------|------|-------|
| `FLASK_SECRET_KEY` | Session 加密密钥 | `dev-change-me` |
| `DATABASE_URL` | SQLite 数据库路径 | `sqlite:///personnel.db` |
| `PUBLIC_BASE_URL` | 系统公网地址（邮件中问卷链接用） | 空（自动推断） |

> **注意**：生产环境务必修改 `FLASK_SECRET_KEY`，并在 `main.py` 中修改默认账户密码。

## 异地情况记录

### 列表查看

侧边栏「异地情况记录」进入已审核的人员异地信息列表。

**列字段**：人员类型、员工编号、姓名、职级、职级日期、工作所在地、工作所在地时间、户口所在地、户口所在地时间、配偶常住地、配偶常住地时间、子女常住地、是否购置房产、房产交付日期、是否全部售出、过渡期截止、附件、异地开始时间、异地结束时间、是否符合异地条件、备注、创建时间。

**交互功能**：
- **列宽拖拽**：拖动表头右边缘调整列宽，刷新后保留（localStorage）。
- **筛选**：点击列头筛选按钮，支持文本筛选（关键词）和日期筛选（从/到）、多选列表筛选。
- **列显示控制**：可隐藏/显示指定列，恢复默认。
- **员工编号双击复制**：双击员工编号单元格自动复制到剪贴板。
- **批量操作**：勾选多条记录后，可批量删除（移至历史）或批量下载附件（ZIP）。
- **导出**：导出当前全量已审核记录，支持 CSV 和 Excel 格式，包含所有 21 个字段。

### 添加记录

侧边栏「添加记录」，手动录入一条异地情况记录。

**必填字段**：
- 人员类型（5 种：中层管理人员、中层管理人员兼省代表、13职级及以上专家岗位人员、省代表、站点总监）
- 员工编号
- 姓名
- 职级（整数）
- 职级日期
- 工作所在地（省 + 市）
- 户口所在地（省 + 市）
- 户口本材料（PDF / JPG / PNG / RAR / ZIP，可多选）
- 是否在工作地购置房产

**选填字段**：
- 配偶常住地（省 + 市）
- 子女常住地（省 + 市）
- 各地时间（工作地时间、户口地时间、配偶地时间）
- 异地开始 / 结束时间
- 备注

**购置房产 = 是时额外必填**：
- 房产交付日期 → 系统自动计算**过渡期截止**（= 交付日期 + 1年 - 1天）
- 是否全部售出
- 房产材料（房产证或购房合同，可多选）

**复制已有记录**：在列表操作列点击「复制」，基于已有记录预填表单（包括附件可选择性复制）。

### 编辑记录

在列表操作列点击「编辑」，进入编辑页面。所有字段可修改，可追加新附件（不覆盖已上传文件），也可单独删除已上传附件。

> 若编辑的是待审核记录，保存后重定向回审核页。

### 审核工作流

侧边栏「待审核」查看所有待审核记录（status = `pending`）。

**来源**：
- 员工通过公开问卷链接提交的记录
- 数据导入时选择「进入审核」模式

**操作**：
- **通过**：记录 status → `approved`，若对应员工编号在员工库中不存在则自动创建员工。
- **拒绝**：记录 status → `rejected`，移至历史记录。
- **批量通过 / 拒绝**：勾选多条后选择操作。
- **一键全部通过**：将全部待审记录一次性通过。

### 历史记录与垃圾桶

**历史记录**（侧边栏「历史记录」）：
- 显示所有 status 为 `rejected` 或 `deleted` 的记录。
- 可「恢复」回已审核状态，或「永久删除」（移至垃圾桶并保存快照）。

**垃圾桶**（侧边栏「垃圾桶」）：
- 显示已永久删除的记录快照（JSON 存储）。
- 可「恢复」重建为已审核记录，或「彻底删除」（不可恢复）。

### 数据导入

侧边栏「数据导入」，支持 Excel（.xlsx）和 CSV 文件批量导入异地记录。

**导入模式**（三选一）：

| 模式 | 说明 |
|-----|------|
| 新增（进入审核） | 导入的记录 status = `pending`，需逐条或批量审核后生效 |
| 新增（直接通过） | 导入的记录 status = `approved`，立即出现在记录列表中 |
| 全量覆盖 | 清空现有全部记录后导入，status = `approved` |

**支持的列名**（中文列头）：

人员类型、员工编号、姓名、职级、职级日期、工作所在地、工作所在地时间、户口所在地、户口所在地时间、配偶常住地、配偶常住地时间、子女常住地、是否在工作地购置房产、房产交付日期、在工作地购置房产是否全部售出、异地开始时间、异地结束时间、备注。

**日期格式**：支持宽松解析，包括：
- `YYYY-MM-DD`、`YYYY/MM/DD`、`YYYY.MM.DD`、`YYYYMMDD`
- `YYYY年MM月DD日`
- `DD-MM-YYYY`、`MM/DD/YYYY`

## 公开问卷链接

为员工生成专属一次性问卷链接，员工通过链接自助填报或更新异地信息，无需登录系统。

### 生成链接

侧边栏「生成记录链接」，输入员工编号后点击「生成链接」：
- 若员工已有审核通过的记录，生成**预填链接**（自动填充最新记录数据）。
- 若员工无记录，生成**空白链接**（员工从零填写）。

生成的链接格式：`http://<系统地址>/form/<token>`，每次生成唯一 token，**每个链接只能提交一次**。

### 问卷填写（员工端）

**首次填写**：填写全部信息后提交，生成一条待审核记录。

**更新地点信息**：系统会自动检测本次提交与最新已审核记录的地点变化：
- 若工作地 / 户口地 / 配偶地发生变更，系统按变更日期**自动拆分为多条待审核记录**，每条对应一个时间节点的地点组合。
- 若仅房产信息变化（无地点变更），生成一条待审记录。
- 拆分规则示例：若员工先变更了户口地（2月1日），再变更了工作地（3月1日），系统生成两条记录，分别反映两个时间点的地点组合，及对应的异地条件状态。

**附件要求**：
- 户口所在地发生变更时，需重新上传户口本材料。
- 选择在工作地购置房产时，需上传房产证或购房合同。

## 员工管理

侧边栏「员工列表」管理员工基本信息库，为邮件发送和问卷预填提供数据支撑。

### 基本操作

| 操作 | 说明 |
|-----|------|
| 添加员工 | 填写员工编号（必填）、姓名（必填）、邮箱（可选） |
| 编辑员工 | 修改姓名和邮箱 |
| 删除员工 | 单条删除 |
| 批量删除 | 勾选多条 → 点击「批量删除」→ 确认弹窗 |

### 批量填充默认邮箱

点击「默认邮箱」按钮，为邮箱为空的员工自动生成邮箱：

**生成规则**：姓名拼音全拼（小写）+ `@cmss.chinamobile.com`

示例：李明 → `liming@cmss.chinamobile.com`

支持两种填充范围：
- **为所有缺失邮箱的员工填充**
- **仅为已勾选的员工填充**（已有邮箱者不覆盖）

### 导入 / 导出

**导出**：支持 Excel 和 CSV，员工编号列保留文本格式（防止 Excel 自动去除前导零）。

**导入**：支持 Excel / CSV，列名为「员工编号」「姓名」「邮箱」，已存在的员工编号跳过（不覆盖）。

### 搜索员工

员工列表和邮件发送页面均支持按员工编号或姓名关键词搜索（调用 `/employees/api/search?q=关键词`）。

## 邮件模板

侧边栏「邮件模板」管理可复用的邮件模板，支持富文本编辑和宏变量替换。

### 模板字段

| 字段 | 说明 |
|-----|------|
| 模板名称 | 仅用于内部标识 |
| 邮件主题 | 支持宏变量 |
| 邮件正文 | 富文本编辑器（支持字体、字号、加粗、颜色、对齐、列表等） |
| 抄送（CC） | 逗号分隔多个邮箱 |
| 密送（BCC） | 逗号分隔多个邮箱 |
| 附件 | 可上传固定附件随邮件一并发送 |

### 宏变量

模板中使用 `{{宏名称}}` 格式插入变量，发送时自动替换为每位收件人的实际数据：

**基础宏（收件员工信息）**：

| 宏 | 替换内容 |
|----|---------|
| `{{姓名}}` | 收件员工的姓名 |
| `{{员工编号}}` | 收件员工的工号 |
| `{{邮箱}}` | 收件员工的邮箱地址 |
| `{{问卷链接}}` | 为每位收件人自动生成专属一次性问卷链接（含历史回填） |

**异地记录宏（取员工最新已审核记录）**：

| 宏 | 替换内容 |
|----|---------|
| `{{人员类型}}` | 人员类型 |
| `{{职级}}` | 职级 |
| `{{职级日期}}` | 职级日期 |
| `{{工作所在地}}` | 工作所在地 |
| `{{工作所在地时间}}` | 工作所在地变更时间 |
| `{{户口所在地}}` | 户口所在地 |
| `{{户口所在地时间}}` | 户口所在地变更时间 |
| `{{配偶常住地}}` | 配偶常住地 |
| `{{配偶常住地时间}}` | 配偶常住地变更时间 |
| `{{子女常住地}}` | 子女常住地 |
| `{{是否在工作地购置房产}}` | 是 / 否 |
| `{{房产交付日期}}` | 房产交付日期 |
| `{{在工作地购置房产是否全部售出}}` | 是 / 否 |
| `{{过渡期截止}}` | 过渡期截止日期 |
| `{{异地开始时间}}` | 异地开始时间 |
| `{{异地结束时间}}` | 异地结束时间 |
| `{{是否符合异地条件}}` | 是 / 否 |
| `{{备注}}` | 备注 |

> 若员工无已审核异地记录，对应宏替换为空字符串。
> `{{问卷链接}}` 在正文中会被渲染为可点击的超链接。

### 使用宏的技巧

编辑模板时，右侧宏列表点击即可插入到当前光标位置（主题框或正文编辑器）。

## 邮件发送

侧边栏「邮件发送」，即时发送邮件给指定员工。

### 操作步骤

1. 在员工列表中搜索并勾选目标员工。
2. 选择邮件模板，界面自动预览该模板的主题和抄送设置。
3. 点击「确认发送」，系统逐一为每位员工生成个性化邮件并发送。

### 发送说明

- 无邮箱的员工会在列表中标注警告，仍可勾选但发送时会失败并记录。
- 发送结果实时显示成功 / 失败数量及错误原因。
- 发送记录自动保存到「发送记录」页面。
- `{{问卷链接}}` 宏：每位收件人的链接独立生成，每个链接只能使用一次。

## 定时发送

侧边栏「定时发送」，配置按计划自动发送邮件的任务，由后台 APScheduler 驱动。

### 计划类型

| 类型 | 说明 |
|-----|------|
| 每天 | 每天指定时刻发送 |
| 每月 | 每月指定日期的指定时刻发送 |
| 单次 | 在指定日期时刻发送一次，执行后自动禁用 |

### 收件人模式

| 模式 | 说明 |
|-----|------|
| 手选员工 | 指定员工 ID 列表 |
| 按条件筛选 | 按职级范围、是否符合异地条件、异地开始时间范围筛选 |

**筛选条件**：
- 职级范围：职级 ≥ 最小值 且 职级 ≤ 最大值
- 异地状态：符合异地条件 / 不符合 / 不限
- 异地开始时间范围：从某日期到某日期之间

### 任务管理

| 操作 | 说明 |
|-----|------|
| 启用 / 禁用 | 切换任务的激活状态（禁用后不再触发） |
| 立即执行 | 无视计划时间，立即执行一次 |
| 编辑 | 修改计划和收件人设置 |
| 删除 | 删除任务并从调度器移除 |

## 邮件配置

侧边栏「邮件配置」配置 SMTP 发件服务器。

### 预设

| 预设 | SMTP 服务器 | 端口 | SSL |
|-----|-----------|-----|-----|
| QQ 邮箱 | smtp.qq.com | 465 | 是 |
| CMSS 企业邮箱 | smtp.chinamobile.com | 465 | 是 |
| 自定义 | 手动填写 | 手动填写 | 可选 |

### 配置字段

| 字段 | 说明 |
|-----|------|
| SMTP 服务器 | 邮件服务器地址 |
| 端口 | 通常 465（SSL）或 587（STARTTLS） |
| 使用 SSL | 开关 |
| 用户名 | 发件邮箱地址 |
| 密码 / 授权码 | SMTP 密码或应用专用授权码 |
| 发件人显示名 | 收件人看到的发件人名称（可选） |

> 可同时保存多个 provider 的配置，每次只有一个处于激活状态。

## 数据备份

侧边栏「数据备份」配置定时将数据自动发送到指定邮箱。

### 备份内容

每次备份发送一封邮件，附带两个附件：
1. **异地情况记录 Excel**：所有已审核记录的完整导出（`异地情况记录_YYYYMMdd_HHmmss.xlsx`）
2. **SQLite 数据库副本**：完整数据库文件（`personnel_YYYYMMdd_HHmmss.db`）

### 计划类型

| 类型 | 可选参数 |
|-----|---------|
| 每天 | 执行时间（HH:MM） |
| 每周 | 周几 + 执行时间 |
| 每月 | 几号 + 执行时间 |

### 配置项

| 字段 | 说明 |
|-----|------|
| 启用备份 | 开关 |
| 计划类型 | 每天 / 每周 / 每月 |
| 执行时间 | HH:MM |
| 收件人 | 多个邮箱用逗号分隔 |
| 邮件主题 | 可自定义（留空则使用默认） |

**立即备份**：点击「立即执行」无视计划时间，立即生成并发送一次备份邮件。

> 备份使用 SQLite Online Backup API 进行安全拷贝，不会影响正在运行的数据库操作。
